3. 앱 종류

애플리케이션은 크게 3가지로 구분된다.

1️⃣ 웹 앱 (Web App)

웹 브라우저에서 실행되는 애플리케이션

HTML / CSS / JavaScript 기반

2️⃣ 하이브리드 앱 (Hybrid App)

웹 기술 + 네이티브 기술 결합

3️⃣ 네이티브 앱 (Native App)

Android / iOS 전용 앱
